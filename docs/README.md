# Sandbox documentation

Sandbox is a self-hosted Rust control plane for disposable coding environments and AI agents.

## Hosted resources

- [Product documentation](https://docs.yshubham.com/v2/sandbox/overview) is the canonical human-facing guide.
- [MCP and agent skills](https://docs.yshubham.com/v2/sandbox/mcp-and-skills) covers coding-agent setup.
- [Tools registry](https://tools.yshubham.com/) hosts verified install scripts and release assets.
- [Forge mirror](https://git.yshubham.com/bas3line/sandbox) mirrors the public source.
- [Platform status](https://status.yshubham.com/) reports public-service health.
